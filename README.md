# AdvancedCalculator - Calculadora Avanzada en Java

## 1. Descripción breve

Este proyecto consiste en una calculadora de consola desarrollada en Java que permite realizar diversas operaciones matemáticas utilizando funciones (métodos). Su objetivo es aplicar conceptos de programación como modularización, reutilización de código y control de flujo.

---

## 2. Tecnologías usadas

- Java 21  
- Scanner  
- Funciones / Métodos  
- Control de flujo (`switch`, `do-while`, `if/else`)  

---

## 3. Cómo ejecutar el proyecto

javac AdvancedCalculator.java
java AdvancedCalculator

---

## 4. Funcionalidades

La calculadora incluye las siguientes operaciones:

- Suma: permite sumar dos números ingresados por el usuario.  
- Resta: calcula la diferencia entre dos números.  
- Multiplicación: realiza la multiplicación de dos valores.  
- División: divide dos números, validando que el divisor no sea cero.  
- Potencia: calcula un número elevado a otro.  
- Raíz cuadrada: obtiene la raíz cuadrada de un número, validando que no sea negativo.  
- Módulo o residuo: calcula el residuo de una división.  
- Salir: finaliza la ejecución del programa.
